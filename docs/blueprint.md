# **App Name**: Greenlab CRM

## Core Features:

- Dashboard: Overview of key metrics, recent activities, and quick access to important modules.
- Chat: Real-time communication platform for internal team collaboration and customer support.
- Contacts: Centralized contact management system to store and organize customer information.
- Leads (Kanban): Visual lead management using a Kanban board to track progress and prioritize tasks.
- Pricing and Costing: Tool to calculate prices of green lab services based on resources, complexity, and other factors. Uses internal database and external web resources to assess pricing ranges. Allows downloading pricing calculations
- Quotes: Generate professional quotes for customers.
- Settings: App settings that allow you to modify the app

## Style Guidelines:

- Primary color: Vibrant green (#90EE90) to reflect 'Greenlab' and growth. Secondary color: Soft gray (#F5F5F5) for a clean and modern look. The choice of green supports a natural and eco-friendly atmosphere.
- Background color: Light gray (#FAFAFA) for the main content area to ensure readability.
- Accent color: Teal (#008080) to highlight important actions and notifications, ensuring visibility and user engagement.
- Font pairing: 'Inter' (sans-serif) for body text providing a clean, modern, neutral look and 'Belleza' (sans-serif) for headlines to lend a fashionable, artistic feel to prominent titles. Longer text: 'Alegreya' (serif) for readability.
- HubSpot-like layout with a sidebar navigation, topbar with search and notifications, and responsive design for different screen sizes.
- Use flat, modern icons from a library like FontAwesome or Material Icons.
- Subtle animations for transitions and loading states to enhance user experience.
